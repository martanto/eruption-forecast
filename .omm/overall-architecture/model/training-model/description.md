Builds labels, extracts features, then runs per-seed `GridSearchCV` fits in parallel via `joblib.Parallel(backend="loky")`. Supports 11 classifier keys (rf, gb, xgb, svm, lr, nn, dt, knn, nb, voting, lite-rf), three CV strategies, and `resample_method="auto|under|over|None"`. After fitting, calls `utils.ml.merge_seed_models` / `merge_all_classifiers` to produce a `SeedEnsemble` per classifier and bundle them into a `ClassifierEnsemble`. Cache identity covers tremor frame + label/feature/fit params + classifier list.
