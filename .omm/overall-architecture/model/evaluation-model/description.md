Dispatches on the upstream stage's `kind`: `training` reuse pulls labels from `TrainingModel.labels`; `prediction` reuse re-builds labels on the prediction window grid using user-supplied `eruption_dates`. Never re-fits a model — `evaluate()` builds a `MetricsEnsemble` once and caches it on `self.MetricsEnsemble` so the per-classifier `predict_proba` pass is amortised across `evaluate()` and `compare()` calls. Writes artefacts under `{output_dir}/evaluation/{kind}/`.
