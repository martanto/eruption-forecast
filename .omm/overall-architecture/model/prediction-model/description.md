Inference stage. Accepts a `ClassifierEnsemble`, `SeedEnsemble`, JSON/PKL registry, or trained-model CSV (resolved via `ClassifierEnsemble.from_any()`). Builds the forecast window grid (`build_label`), extracts features (`extract_features`), then writes per-seed + consensus probabilities under `prediction/results/` and a forecast plot under `prediction/figures/`. Cache identity includes the upstream `training_hash`, so re-training invalidates downstream forecasts.
