Top-level orchestrator. Exposes fluent stage methods `calculate() / train() / predict() / evaluate()` that each return `Self`, plus `save_config()` / `from_config()` / `run()` for round-trip replay. Captures every stage's kwargs into the matching `ForecastConfig` sub-config before defaults are resolved so a saved YAML/JSON config reproduces the run exactly. Holds the active stage instances on attributes named after the class (`fm.TrainingModel`, `fm.PredictionModel`, ...).
