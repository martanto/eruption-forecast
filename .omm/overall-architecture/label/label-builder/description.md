Global sliding-window label builder. Constructs windows via `utils.window.construct_windows`, marks each window erupted=1 when its end falls within `[eruption_date - day_to_forecast, eruption_date]`. `include_eruption_date=False` (default) additionally marks the eruption day positive (total positive span = `day_to_forecast + 1` days). Saves to a filename that encodes start/end, window size, step, and `day_to_forecast` so downstream stages can parse params back.
