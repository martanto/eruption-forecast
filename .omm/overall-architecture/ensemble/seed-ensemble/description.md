sklearn `BaseEstimator + ClassifierMixin`. Bundles all seed estimators for one classifier plus each seed's selected `feature_names`. Factories: `from_registry(csv)`, `from_json(json)`, `from_any(path|obj)`. Inference: `predict_proba(X)`, `predict_with_uncertainty(X, threshold)` aggregate across seeds. Also implements `_VisualBlock` so it renders nicely as an HTML repr in notebooks.
