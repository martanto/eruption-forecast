Per-seed metric engine over a fitted `ClassifierEnsemble`. `compute()` materialises `(n_samples, n_seeds)` `y_proba` / `y_pred` matrices per classifier and persists them under `classifiers/{clf}/predictions/` — no JSON, no per-seed metric files. `plot_aggregate()` renders ROC/PR/threshold/g-mean/MCC plots; `plot_seed()` renders per-seed plots. `save()` joblib-dumps the whole instance so `load()` skips recomputation.
