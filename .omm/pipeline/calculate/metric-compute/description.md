Worker-side compute. The bandpass filters use `BANDPASS_FILTER_CORNERS` corners and `DEFAULT_FREQUENCY_BANDS` bands. `RSAM` and `DSAR` operate over each band; `ShannonEntropy` operates on the unfiltered broadband signal. `utils.window.calculate_window_metrics` does the per-10-minute aggregation. Optional anomaly removal (`utils.dataframe.remove_anomalies`) and outlier removal run before the per-day CSV is written.
