`TrainingModel.build_label()` instantiates either `LabelBuilder` (global) or `DynamicLabelBuilder` (per-eruption, default for overlapping eruptions). The builder constructs the window grid (`utils.window.construct_windows`), marks each window erupted=1 when its end falls within `[eruption_date - day_to_forecast, eruption_date]`, and writes a label CSV whose filename encodes start/end, window size, step, and `day_to_forecast`.
