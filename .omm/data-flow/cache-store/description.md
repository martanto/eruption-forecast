`{station_dir}/cache/{ClassName}/{hash}.pkl` + `{hash}.params.json`. Hash is computed from `CacheModel.build_cache_identity()` which captures every input that affects the stage's output (tremor frame fingerprint, label/feature/fit params, classifier list, and — for prediction — the upstream `training_hash`). `ForecastModel.train()` / `.predict()` opt-in via `use_cache=True` (default).
